## 🛠️ Project 
<h3>Pandas Web-Catcher</h3>
Dự án được xây dựng trên ngôn ngữ lập trình bậc cao Python,
kết hợp khả năng phân tích cấu trúc DOM của BeautifulSoup4
và thư viện Pandas. Điểm nổi bật của công cụ là cơ chế Auto-Discovery.
tự động trích xuất, định dạng lại số liệu (loại bỏ dấu phẩy, xử lý ô trống)
và xuất thẳng ra file .csv chuẩn UTF-8.
